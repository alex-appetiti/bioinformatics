# bioinformatics
Clojure solutions to Rosalind.info problems

To run:
```bash
cat <path-to-data> | clojure -Arun <problem-code>
```

Problem codes are as seen on http://rosalind.info/problems/list-view/ .

### Implemented so far
- DNA
- RNA
- FIB
- REVC
- GC
- HAMM
- PROT
- SUBS
- MRNA
- PERM
- PRTM
- REVP
